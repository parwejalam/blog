Hello, world — why I started this blog

angularmetaJul 14, 20261 min read

Welcome. I'm Parwej, an Angular developer learning AI automation and building things in the open. This is where I write about what I learn.

What to expect

  • Angular and TypeScript notes from real work
  • AI automation experiments (n8n, LLMs, small tools)
  • The occasional deep dive when something finally clicks

How this blog is built

It's an Angular app, but every post is prerendered to static HTML at build time and hosted on GitHub Pages. That means fast first paint and real SEO — the article text is in the HTML before any JavaScript runs.

Posts are plain Markdown files. Adding a new one is just dropping a .md file in and rebuilding.
